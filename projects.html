AudioTag

NFC Audio Tag Management for Android

GPL-3.0 Android Accessible Privacy-First

Transform any NFC tag into a personal audio note. AudioTag enables you to record voice messages, convert text to speech, and create accessible audio experiences with full screen reader support.

  • Storage container contents

Enhance navigation and information access for users with visual impairments. Create audio waypoints and information markers throughout any space.

Technical Specifications

System Requirements

  • Platform: Android 7.0 (API level 24) or higher
  • NFC: NFC-enabled Android device
  • Storage: Approximately 15 MB app size
  • Permissions: NFC access, audio recording, storage access

Audio Specifications

  • Recording Format: AAC audio codec
  • Sample Rate: 44.1 kHz
  • Bit Rate: 128 kbps
  • Maximum Length: 10 minutes per recording

NFC Compatibility

  • Tag Types: NDEF-compatible NFC tags
  • Supported Standards: ISO 14443 Type A/B
  • Memory Requirements: Minimum 96 bytes per tag
  • Range: Standard NFC range (1-4 cm)

Accessibility Features

  • Screen Readers: Full TalkBack support
  • Navigation: Voice commands and gesture navigation
  • Visual: High contrast themes, scalable text
  • Audio: Audio feedback for all interactions
